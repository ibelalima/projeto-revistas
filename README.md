# Projeto Revistas com Iframe 📚

Este repositório contém um projeto de site estático desenvolvido como parte de uma **atividade acadêmica** da faculdade de Engenharia de Software.

## 📝 Sobre o Projeto
O objetivo da atividade foi praticar a estruturação de layouts utilizando tabelas (`<table>`) para organização de menus laterais e a manipulação de `<iframe>` para exibição de conteúdo dinâmico dentro da mesma página.

O site simula uma banca de revistas virtual, onde o usuário pode selecionar diferentes edições no menu lateral e visualizá-las no frame central.

## 🛠️ Tecnologias Utilizadas
* **HTML5**: Estruturação semântica e uso de frames.
* **CSS3**: Estilização com gradientes, sombras e fontes externas (Google Fonts).
* **Google Fonts**: Utilização da fonte 'Playfair Display' para um visual editorial.

## 📁 Estrutura de Arquivos
* `index.html`: Página principal com o layout de tabelas e o iframe.
* `capa.html`: Conteúdo inicial exibido no frame.
* `/revistas`: Pasta contendo os arquivos HTML de cada edição e o CSS específico.
